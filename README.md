# automatisation-ventes
# Analyse des Ventes E-commerce

##  Description

Ce projet Python permet d’automatiser l’analyse des ventes d’une entreprise e-commerce à partir d’un fichier CSV.

Il remplace les tableurs classiques en offrant :

*  des calculs rapides
* une visualisation claire
* une exportation automatique des résultats



## Technologies utilisées

*  Python 3
*  pandas
*  matplotlib



##  Structure du projet


📦 projet
 ┣  ventes.csv
 ┣  script.py
 ┣  resultats_final.csv
 ┗  ca_par_produit.png




##  Fonctionnalités

 Calcul du **Chiffre d’Affaires Brut**
 Calcul du **CA Net après remise**
 Calcul de la **TVA (20%)**
 Identification du produit le plus performant
 Statistiques (moyenne, top 3 produits)
 Génération automatique d’un fichier de résultats
 Visualisation graphique du CA par produit



##  Résultats générés

Après exécution :

*  `resultats_final.csv` → données complètes avec calculs
*  `ca_par_produit.png` → graphique du CA Net



##  Objectif du projet

Mettre en pratique les compétences en Python, manipulation de données et visualisation avec des outils professionnels utilisés en entreprise.
